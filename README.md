# 📚 Library Intelligence Project

An end-to-end data analytics project built for the library at **Marathwada Mitra Mandal's College of Engineering (MMCOE), Pune**.

 📌 Objective
To analyze library visitor patterns and book collection data to help the institution make informed decisions about resource allocation, collection development, and student engagement.

 📊 Key Metrics

|        Metric          | Value                 |
|------------------------|-----------------------|
| Total Library Visits   | 56,811                |
| Unique Book Titles     | 3,747                 |
| Total Book Copies      | 26,598                |
| Total Collection Value | ₹4.82 Crore           |
| Top Department         | Computer Engineering  |
| Top Domain             | Computer Science & IT |
| Analysis Period        | Nov 2023 – Mar 2026   |


 📂 What's Inside

 📊 Dashboard
Tabs included:
- Monthly Visits — Visitor trends from Nov 2023 to Mar 2026
- By Department  — Which departments visit the most
- Book Analysis  — Collection breakdown by domain and sub-category
- Copies & Cost  — Searchable book-level pricing table
- Top Authors    — Most represented authors in the collection
- SQL View       — Live query results with domain/sub-category filters

 🔢 Project Highlights
- 56,811 total library visits analyzed
- 3,747 unique book titles across 11 domains
- 26,598 total copies worth ₹4.82 Crore
- Data span: November 2023 – March 2026

 🛠️ Tools Used
- Microsoft Excel — data cleaning and preparation
- SQL — querying and aggregation
- HTML + Chart.js — interactive dashboard
